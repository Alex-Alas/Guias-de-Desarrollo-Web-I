Datos Generales de la Materia

Descripción del Curso:

En este curso, los estudiantes aprenderán los fundamentos de HTML, CSS y JavaScript para crear aplicaciones web funcionales e interactivas. Se cubrirán temas cómo: estructurar páginas web con HTML, aplicar estilos mediante CSS, implementar interactividad con JavaScript moderno, comprender la arquitectura front-end y las mejores prácticas de desarrollo. La metodología combina lecturas previas, sesiones prácticas presenciales y un proyecto grupal con retroalimentación continua que permitirá a los estudiantes dominar las tres tecnologías fundamentales del desarrollo web.

Contenidos del Curso:

  • Elementos de lenguaje HTML para elaborar la estructura de una aplicación web
  • Propiedades y selectores utilizados en hojas de estilo en cascada (CSS) para dar estilo a una página web.
  • Elementos que permiten manejar la interacción del sistema con el usuario de acuerdo con elementos de sintaxis moderna en Javascript.

Guías de Ejercicios

Guía 1: Introducción a HTML y CSS

Esta guía cubre los fundamentos de HTML para la estructuración de contenido y CSS para la estilización básica, incluyendo estilos en línea y embebidos.

  • Ejercicio de Estilos Embebidos
  • Ejemplo 2
  • Ejercicio de Países
  • Mi Primera Página

Guía 2: Formularios y Tablas en HTML

En esta guía se exploró la creación de formularios para la captura de datos del usuario y el uso de tablas para mostrar información de manera estructurada.

  • Ejercicio de Película
  • Ejercicio de Tablero
  • Mi Biografía

Guía 3: CSS Avanzado y Layouts

Esta guía se enfocó en técnicas más avanzadas de CSS, como la creación de menús de navegación horizontales y el análisis de diseños web existentes.

  • Ejercicio de los Óscares
  • Menú Horizontal
  • Análisis de Diseño
  • Agencia de Viajes (Layout)

Guía 4: Diseño Responsivo con Media Queries

El objetivo de esta guía fue aprender a crear sitios web adaptables a diferentes tamaños de pantalla utilizando Media Queries en CSS.

  • Ejercicio de Media Queries
  • Agencia de Viajes v2 (Responsiva)

Análisis FODA

Fortalezas

¿Qué habilidades he dominado o mejorado significativamente?

He mejorado mi capacidad para estructurar contenido con etiquetas semánticas de HTML y aplicar reglas de estilos precisas con CSS. Además, domino la creación de formularios, el uso de tablas y la implementación de layouts flexibles y responsivos.

Oportunidades

¿Qué he aprendido que me pueda servir en otros proyectos o ámbitos profesionales?

Experiencia con HTML y CSS, pues son la base del desarrollo front-end. Lo aprendido sobre diseño responsivo con media queries y Flexbox es crucial para crear experiencias de usuario modernas y accesibles en cualquier dispositivo, lo cual tiene alta demanda en la industria a pesar de la variedad de herramientas de Inteligencia Artificial (AI por sus siglas en inglés).

Debilidades

¿Qué áreas necesito reforzar o en cuáles me gustaría profundizar?

Aunque he construido layouts estáticos y responsivos, necesito profundizar en JavaScript y sus usos para crear aplicaciones web más dinámicas. También, considero que podría explorar frameworks de CSS como Tailwind para agilizar el desarrollo y diseñar interfaces más complejas y atractivas.

Amenazas

¿Qué obstáculos encontré (personales o técnicos) que podría haber impactado mi progreso?

Un obstáculo técnico común es la compatibilidad entre navegadores; asegurar que un sitio se vea y funcione igual en todos ellos requiere pruebas constantes. Personalmente, la gestión del tiempo para depurar errores en el código, especialmente en CSS, puede ser un desafío que impacta la velocidad del desarrollo.
